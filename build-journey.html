The Build Timeline

Five weeks, five layers — each building on the last

Week 1 · Feb 7–10 · Foundation
From Zero to Working Product
Stood up the entire architecture in 4 days — React 19 SPA with Firebase auth, branded editorial design system, 8 interactive learning modules, the Academic Journey Simulator (a branching-narrative game with 12 archetypes), and a CAO Points Simulator. Deployed to Firebase Hosting on day 3.
React 19 + TypeScript Firebase Auth 8 Modules Journey Simulator CAO Calculator Landing Page
Week 2 · Feb 11–12 · Depth
Gamification & Engagement Layer
Built the full reward economy — points, 8 athlete ranks, study streaks, a Deep Focus Timer with ambient sounds, and an island customisation shop with 85 unlockable items. Redesigned every interactive component across all modules for consistency and polish.
Points Engine 8 Ranks Streaks 85 Shop Items Focus Timer Component Redesign
Week 3 · Feb 17–20 · Scale
Teacher Tools & 29 Subject Guides
Launched the Guidance Counsellor dashboard (per-school student views, alert system, radar charts), built the North Star onboarding system with 6 motivation archetypes, added 29 subject-specific exam guides covering every LC subject, and shipped a full mobile-first UX overhaul.
GC Dashboard North Star Onboarding 29 Subject Guides Mobile UX Alert System 17 Bug Fixes
Week 4 · Mar 8–9 · Intelligence
Adaptive Scheduling & Strategy
Implemented the SM-2 spaced repetition algorithm for personalised timetables with weekend-aware loading and daily caps. Built the War Room (strategic study command centre), Comeback Engine (quickest-win analysis), peer social features, and full study session tracking with debrief system.
SM-2 Algorithm War Room Comeback Engine Social Features Study Sessions Debrief System
Week 5 · Mar 11–13 · Integration
Connected Intelligence
Shipped Future Finder (CAO course matching), Syllabus X-Ray (mark-weight analysis), Learning DNA (strategy effectiveness tracking), and then wired 14 cross-tool integrations — so Future Finder informs the Comeback Engine, debriefs auto-seed the War Room, journey archetypes personalise study tips, and simulator scenarios generate missions.
Future Finder Syllabus X-Ray Learning DNA 14 Integrations Cross-Tool Data Flow Module Audit
